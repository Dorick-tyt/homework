# Маскировка и форматирование

Набор функций для обработки персональных данных: маскировки номеров карт и счетов, а также преобразования форматов даты.

## Функционал

Проект включает две основные функции:

1. **`mask_account_card()`** — маскирует номера банковских карт или счетов в строке.
2. **`get_date()`** — преобразует дату из формата ISO в формат «ДД.ММ.ГГГГ».

## Использование

### Маскировка номера карты или счёта

Функция `mask_account_card(card_number)` принимает строку с номером карты/счёта и возвращает замаскированную версию.

**Параметры:**
* `card_number` (`str`) — строка, содержащая тип карты/счёта и номер (например, `"Visa Platinum 7000792289606361"`).

**Возвращает:**
* Строку с замаскированным номером в формате `"<тип> <замаскированный номер>"`.
* `None`, если входные данные некорректны или произошла ошибка маскировки.

**Примеры:**
```python
print(mask_account_card("Visa Platinum 7000792289606361"))
# Вывод: Visa Platinum 7000****6361

print(mask_account_card("Счёт 40817810099910004312"))
# Вывод: Счёт 4081****4312

print(mask_account_card("Invalid input"))
# Вывод: None
