# Processing Module

Модуль `processing` предоставляет функции для фильтрации и сортировки списков словарей по заданным критериям. Предназначен для работы с данными, содержащими поля `state` и `date`.

## Установка

Модуль не требует отдельной установки — достаточно разместить файл `processing.py` в проекте.

## Использование

Импортируйте функции из модуля:

```python
from processing import filter_by_state, sort_by_date
```
## Тестирование

Модуль сопровождается набором тестов для проверки корректности работы функций.

### Требования для запуска тестов

* Python 3.8+;
* pytest;
* pytest-cov (для измерения покрытия кода);
* pytest-html (для генерации HTML‑отчётов).
