# Проект: Generators — набор генераторов для работы с финансовыми данными

Этот проект содержит набор юнит‑тестов для проверки корректности работы функций, обрабатывающих данные о банковских транзакциях и генерирующих номера карт.

## Установка

Проект не требует отдельной установки — достаточно скопировать файлы в ваш проект или импортировать модули напрямую.

**Зависимости:** Python 3.6+ (стандартная библиотека, дополнительных зависимостей нет).

## Использование

## Часть 1. Функции модуля `generators.py`

### 1. `filter_by_currency(transactions, currency_code)`

**Назначение:** фильтрация списка транзакций по коду валюты.

**Параметры:**
- `transactions` (`list`) — список словарей с данными транзакций;
- `currency_code` (`str`) — код валюты для фильтрации (например, `"USD"`, `"EUR"`).

**Возвращает:** генератор (`generator`) отфильтрованных транзакций.

**Логика работы:**
- проверяет поле `operationAmount.currency.code` в каждой транзакции;
- возвращает только транзакции с точным совпадением кода валюты.

**Пример использования:**
```python
usd_transactions = list(filter_by_currency(transactions, "USD"))
```
### 2. transaction_descriptions(transactions)
**Назначение:** извлечение описаний операций из списка транзакций.

**Параметры:**

- `transactions` (`list`) — список словарей с данными транзакций.

**Возвращает:** генератор (`generator`) строк‑описаний транзакций.

**Логика работы:**

- извлекает значение поля `description` из каждой транзакции;

- пропускает транзакции без поля description.

### 3. card_number_generator(start, end)
**Назначение:** извлечение описаний операций из списка транзакций.

**Параметры:**

- `start` (`int`) — список словарей с данными транзакци.
- `end` (`int`) — конечное число диапазона.

**Возвращает:** генератор (`generator`)  строк с номерами карт в формате XXXX XXXX XXXX XXXX.

**Логика работы:**

- автоматически корректирует границы:
  - если `start < 1`, берёт `1`;
  - если `end > 9999999999999999`, берёт максимальное значение;
- дополняет числа нулями до 16 знаков;
- разделяет номер на группы по 4 цифры.

## Тестирование

#### `TestFilterByCurrency`
Проверяет фильтрацию транзакций по валютам:
* фильтрация по существующим валютам (`USD`, `EUR`);
* отсутствие совпадений (`RUB`);
* обработка пустого списка транзакций;
* обработка транзакций без поля `operationAmount`.

#### `TestTransactionDescriptions`
Проверяет извлечение описаний транзакций:
* нормальный случай (транзакции с описаниями);
* пустой список транзакций;
* транзакции без описаний;
* смешанный случай (есть и нет описаний);
* одна транзакция с описанием.

#### `TestCardNumberGenerator`
Проверяет генератор номеров карт:
* небольшой диапазон номеров;
* один номер;
* минимальное и максимальное значения;
* граничные случаи (`start < 1`, `end > max`);
* большой диапазон (проверка первых элементов).
